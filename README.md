## Info
Ein Alarm- & Einsatzmonitor für die Feuerwehr. 
Inklusive Personal, Wetter, Einsatz und Fahrzeugstatus. </br>
Aktuell nur für den spezifischen Zugang mit einem GoogleSheet, theoretisch wenn man die .js Abfragen anpasst aber für jedes GoogleSheet möglich.

## Features

- Uhr
- Dienstplan
- Personal
- Einsatzübersicht
- Fahrzeugstatus
- Wetter
- Data from a Google Sheet

</br>
